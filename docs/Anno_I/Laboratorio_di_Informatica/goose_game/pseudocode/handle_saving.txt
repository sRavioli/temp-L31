FUNZIONE save_game()
====================

Questa funzione scrive su file binario una partita in corso.

INPUT:
- players, struttura con i giocatori del gioco, Players;
- board, tabellone di gioco, Board;

OUTPUT:
- nullo, non ha output.

LAVORO:
- game, struttura di salvataggio del gioco, Game;


ALGORITMO:
SE (get_num_games(games) < MAX_GAMES) // 5
  ALLORA chedere all'utente che partita eliminare // choose_saved_game()
  ALTRIMENTI
    scrivere normalmente nella struct (num_games + 1)
    game := set_game_players(game, players)
    game := set_game_board(game, board)

    game_name := LeggereDaTastiera()
    game := set_game_name(game, game_name)
    str_concat(game_name, ".bin")

    ScrivereSuFile(game, game_name)
FINE
