DICHIARAZIONE DELLE FUNZIONI PRINCIPALI PER IL MENÙ DI GIOCO
============================================================

Questo file contiene le dichiarazioni delle funzioni del menù principale di 
gioco.

FUNZIONE print_menu(filename)
-----------------------------

Stampa il menu che è contenuto all'interno di filename.

INPUT:
- filename, il nome del file da cui leggere il menù da stampare, file.

OUTPUT:
- nullo, non restituisce nulla.


MENÙ PRINCIPALE
===============

FUNZIONE main_menu()
--------------------

Stampa il menù principale contenuto in un file di testo apposito utilizzando la 
funzione print_menu(), dopodiché fa scegliere all'utente la voce di menù che 
desidera.

INPUT:
- nullo, non riceve nulla.

OUTPUT:
- nullo, non restituisce nulla.

FUNZIONE print_menu()
---------------------

Stampa a video il menù principale. Esso è composto da:
- nuova partita;
- carica partita;
- classifica;
- aiuto;
- esci.

INPUT:
- nullo, non riceve input.

OUTPUT:
- nullo, non riceve output.


MENÙ NUOVA PARTITA
==================

FUNZIONE print_new_game_menu()
------------------------------

Stampa a video il menù "nuova partita". Esso è composto da:
- inizia partita;
- salva partita;
- abbandona partita;
- esci.

INPUT:
- nullo, non riceve input.

OUTPUT:
- nullo, non riceve output.


FUNZIONE start_new_game()
-------------------------

Stampa a video le opzioni che i giocatori possono scegliere, le quali sono:
- numero di giocatori;
- lunghezza del tabellone;
È sempre presente l'opzione esci.
Dopodiché avvia la partita.

INPUT:
- nullo, non riceve input.

OUTPUT:
- nullo, non riceve output.


FUNZIONE save_game()
--------------------

Chiede all'utente tramite la FUNZIONE get_save_file_name() il nome del file di 
salvataggio e successivamente salva la partita in corso.

INPUT:
- file, file contente la partita da salvare, FILE.

OUTPUT:
- nullo, non restituisce nessun tipo di dato.

FUNZIONE get_save_file_name()
-----------------------------

Chiede all'utente il nome con cui salvare la partita in corso.

INPUT:
- nullo, non riceve nessun tipo di dato.

OUTPUT:
- filename, file contente la partita salvata dell'utente, FILE.

FUNZIONE leave_game()
---------------------

Se l'utente è in partita, chiede all'utente se la voglia salvare, se sì chiama
la funzione save_game(), altrimenti torna al menù principale senza salvare.

INPUT:
- nullo, non riceve nessun tipo di dato.

OUTPUT:
- nullo, non restituisce nessun tipo di dato.


MENÙ CARICA PARTITA
===================

FUNZIONE print_load_game_menu()
-------------------------------

Stampa a video i file salvati (massimo cinque) e chiama choose_saved_file().
È presente il tasto esci, che fa tornare al menù principale.

INPUT:
- nullo, non riceve nessun tipo di dato.

OUTPUT:
- nullo, non restituisce nessun tipo di dato.


FUNZIONE choose_saved_file()
----------------------------

Chiede all'utente quale file vuole scegliere tra quelli salvati.
È presente il tasto esci, che fa tornare al menù principale.

INPUT:
- nullo, non riceve nessun tipo di dato.

OUTPUT:
- filename, file scelto dall'utente che dev'essere caricato, FILE.


FUNZIONE load_game()
--------------------

Carica una partita da un file di testo.

INPUT:
- file, file contenente le opzioni e la partita scelta dall'utente, FILE.

OUTPUT:
- nullo, non restituisce nessun tipo di dato.
